# UserScripts
###### Made by @grajef
Script titles can be clicked to get the install URL

# [AFTLite Tab Names](https://github.com/JGray0705/UserScripts/raw/master/AFTLiteTabTitles.user.js)
### Modifies the tab names of AFTLite pages to make them easier to find. (No more clicking through 50 tabs to find the right one)
![tabs renamed](https://i.imgur.com/eGJnFta.png)

# [ASM Indirect](https://github.com/JGray0705/UserScripts/raw/master/ASMIndirect.user.js)
### Adds buttons to the indirect action page to sign the current user into either ASM or EOS. Lines 13 and 14 can be modified to change the action if you want something else.

# [Blind Counts Tools](https://github.com/JGray0705/UserScripts/raw/master/BlindCountsDueDate.user.js)
### Adds a row to show when each count list is due to be compliant. Also shows the last action of the person assigned to the list and highlights them if they are EOS
![blind counts page](https://imgur.com/mNbhUAG.png)

# [Copy Labor Summary](https://github.com/JGray0705/UserScripts/raw/master/CopyLaborSummary.user.js)

### Highlight and copy the labor summary table

# [Daily Detail Help](https://github.com/JGray0705/UserScripts/raw/master/DailyDetailHelp.user.js)

### Adds a lookup history link to the usernames on the daily detail page.

# [Date Input Helper](https://github.com/JGray0705/UserScripts/raw/master/DateInputHelper.user.js)
![date input buttons](https://imgur.com/oZItMR5.png)

### Adds buttons to select time frames on UPH Drilldown, Daily Detail, and Labor Summary pages.

# [Find People Tools](https://github.com/JGray0705/UserScripts/raw/master/FindPeopleTools.user.js)

### Adds a 30 second (togglable) auto refresh to the find people page, highlights idle logins in red, and adds an indirect action form to the top of the find people page.

![indirect action on find people](https://imgur.com/fjVOVDn.png)
![highlighted idle time](https://imgur.com/tkyWJTU.png)

# [Inventory Add-On](https://github.com/JGray0705/UserScripts/raw/master/InventoryAddOn.user.js)

### Moves the inventory search form to the bottom of every inventory page (removes the need to go back a page to search another item/location). Also includes a picture of the ASIN if available
![Inventory update](https://imgur.com/HG8Q0Jb.png)

# [NYR Dashboard Colors](https://github.com/JGray0705/UserScripts/raw/master/NYRDashboardColors.user.js)

### Colors SLA times on the NYR dashboard page along with adding a 30 second auto refresh.

# [Pallet Counter](https://github.com/JGray0705/UserScripts/raw/master/PalletCounter.user.js)

### Shows the number of pallets in a tranship load.
